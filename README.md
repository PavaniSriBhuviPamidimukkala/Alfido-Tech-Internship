# Breast Cancer Classification Using Machine Learning

## Project Overview

This project implements supervised machine learning classification using Logistic Regression and Random Forest algorithms to predict whether a breast tumor is malignant or benign.

The project demonstrates:
- Data preprocessing
- Train-test split
- Feature scaling
- Cross-validation
- Model comparison
- Performance evaluation
- Data visualization

---

## Dataset

The project uses the Breast Cancer Wisconsin dataset available in the scikit-learn library.

Dataset Details:
- Total Samples: 569
- Features: 30 numerical features
- Classes:
  - Malignant
  - Benign
